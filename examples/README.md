# Archivos de Ejemplo

Esta carpeta contiene archivos de ejemplo que puedes usar como referencia o para importar datos de prueba.

## Estructura

- `design-metrics-example.json` - Métricas de diseño de ejemplo (Figma)
- `development-metrics-example.json` - Métricas de desarrollo de ejemplo (GitHub)
- `roi-example.json` - Ejemplo de cálculo de ROI
- `okr-example.json` - Ejemplo de OKRs
- `complete-sample.json` - Dataset completo de ejemplo

## Uso

Estos archivos pueden usarse para:
1. Entender la estructura de datos esperada
2. Probar la importación de datos manuales
3. Usar como templates para crear nuevos datos

## Cómo importar

Puedes copiar el contenido de estos archivos y enviarlos mediante:
- POST `/api/metrics/manual` para métricas
- POST `/api/roi` para ROI
- POST `/api/okrs` para OKRs
